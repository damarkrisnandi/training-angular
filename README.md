# Training Angular

## Kenapa dan Apa Angular?
* Kerangka kerja untuk membangun aplikasi sisi klien.
* Cocok untuk membuat web SPA (Single Page Applications).
* Modular.
* Re-usable Code.
* Fast & Easy Development.
* Unit testable.
* Google + Microsoft.
